# Plan de Negocio - Empresa de Desarrollo Web

## Nombre: DevPerú Studio

---

## 1. Resumen Ejecutivo

Desarrollamos sitios web y aplicaciones web para empresas y emprendedores en Perú. Entregamos soluciones digitales de calidad a precios accesibles.

---

## 2. Servicios

| Servicio | Descripción | Precio Base (S/) |
|----------|-------------|------------------|
| Landing Page | Página única para captar clientes | 150 - 200 |
| Sitio Web Corporativo | 5 páginas, diseño responsive | 280 - 350 |
| E-commerce | Tienda online básica | 400 |
| App Web a Medida | Sistema a medida (CRMs, dashboards) | 400 |
| Mantenimiento | Soporte mensual | 50 - 100/mes |

---

## 3. Público Objetivo

- Empresas pequeñas y medianas (MYPE)
- Emprendedores que necesitan presencia online
- Profesionales independientes (abogados, doctores, etc.)
- Negocios locales (restaurantes, tiendas)

---

## 4. Ventajas Competitivas

- Precios accesibles comparado con agencias grandes
- Entrega rápida (1-3 semanas según proyecto)
- Comunicación directa con el desarrollador
- Soporte post-venta incluido

---

## 5. Canales de Venta

- Instagram y Facebook
- LinkedIn
- MercadoLibre (servicios)
- Referidos de clientes satisfechos
- Google My Business

---

## 6. Inversión Inicial Estimada

| Concepto | Costo (S/) |
|----------|------------|
| Registro de empresa (SAS) | 500 - 1,000 |
| Computadora/Equipo | 2,000 - 3,000 |
| Internet | 150/mes |
| Dominio + Hosting | 200/año |
| Marketing inicial | 300 |
| **Total** | **3,150 - 4,650** |

---

## 7. Meta a 6 Meses

- Conseguir 10-15 clientes
- Generar S/ 2,000 - 4,000 mensuales
- Construir portafolio con proyectos reales

---

## 8. Stack Tecnológico Recomendado

- **Frontend:** HTML, CSS, JavaScript, React
- **Backend:** Node.js, Python (Django/Flask), PHP
- **Hosting:** Vercel, Netlify, Hostinger
- **Base de datos:** PostgreSQL, MongoDB
- **Control de versiones:** GitHub
